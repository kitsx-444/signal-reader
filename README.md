# signal-reader

### What is this program:
This is a signal reader. It gets the data from two imported csv files and uses conditionals to give either a Buy, Sell or Watch signal.

### Concepts used
1. Pandas
2. CSV reading
3. Lambda
4. apply()
5. File merging
6. axis=1

### What I learned:
axis=1 tells python to run over the table row by row (side by side) and axis=0 tells python to run over a whole column (top down)
